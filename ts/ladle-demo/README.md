# Ladle demo

## Prerequisites

If not pnpm executables directory initialised:

```bash
pnpm setup
```

## Installation

```bash
pnpm install
pnpm run ladle
```

## Generate story with controls

Using [ladlescoop](https://github.com/lukifer/ladlescoop):

```bash
pnpm --global install ladlescoop
pnpm exec ladlescoop src/button.tsx
```
